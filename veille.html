01C'EST QUOI LE ZERO TRUST ?
Fevrier 2025// kiteworks.com
Le nouveau perimetre : l'identite

Pendant des annees, la securite IT reposait sur un principe simple : tout ce qui est a l'interieur du reseau est de confiance. Le pare-feu protegeait le chateau. Avec le teletravail, le cloud et les appareils personnels, ce modele est devenu obsolete.

Le Zero Trust renverse tout : ne jamais faire confiance, toujours verifier. Chaque utilisateur, chaque machine, chaque application doit prouver son identite a chaque acces, meme depuis l'interieur du reseau.

02LA FIN DES VPN : CHIFFRES 2025
Avril 2025// Zscaler ThreatLabz + CIO.com
VPN Risk Report 2025 : bascule massive vers le Zero Trust

Le rapport Zscaler (600+ pros IT interroges) est sans appel. Les CVE VPN ont augmente de 82,5% en un an. La CVE-2025-22457 permet a un attaquant non authentifie d'executer du code a distance. 92% des entreprises redoutent que des failles VPN non patchees menent a des ransomwares.

La solution : le ZTNA (Zero Trust Network Access), qui remplace le tunnel VPN par un acces granulaire verifie en permanence, application par application.

03ZERO TRUST ET LES METIERS SISR
Decembre 2025// lockself.com + ANSSI
Les outils concrets : AD, VLAN, Palo Alto, GLPI

Mettre en place le Zero Trust ne signifie pas tout racheter. Les techniciens SISR travaillent deja avec les briques fondamentales :

-- Active Directory + MFA : verification d'identite a chaque acces sensible
-- Micro-segmentation VLAN : isoler les environnements par service
-- Pare-feu NGFW Palo Alto : filtrage applicatif et controle des flux entre segments
-- GLPI + supervision : inventaire continu et traitement des incidents

Zero Trust pour les PME : indispensable en 2025

En 2024, plus de 60% des cyberattaques en France ont vise des PME (ANSSI). Avec le teletravail devenu norme, les frontieres reseau ont disparu. Une PME peut commencer progressivement : renforcer l'authentification, puis segmenter le reseau, puis mettre en place une supervision continue.
